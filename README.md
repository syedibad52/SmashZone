# 🏸 SmashZone — Badminton & Fitness Website

SmashZone is a modern and responsive badminton & fitness website built using **HTML, CSS, and JavaScript** in a single file structure.  
The project focuses on providing badminton fitness information, calorie tracking, beginner guides, health benefits, and interactive UI elements with a clean premium design.

---

## 🚀 Features

- 🎨 Modern responsive UI design
- 🏸 Badminton fitness and training guide
- 🔥 Interactive calorie burn calculator
- 📊 Dynamic fitness tracker
- 💪 Muscle activation section
- 📱 Fully responsive for mobile and desktop
- ⚡ Smooth animations and hover effects
- 🎯 Beginner-friendly badminton tutorials
- 🧠 Health and fitness tips
- 🌈 Elegant color palette and typography

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript**
- **Tabler Icons**
- **Google Fonts**
